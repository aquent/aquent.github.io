Buttons

To render button styles, a "button" class is required, even on 'button' elements.








                  
<button type="button" name="button">unstyled button element</button>
<button class="button" type="button" name="button">.button (.button.primary)</button>
<button class="button secondary" type="button" name="button">.button.secondary</button>
<button class="button dark-gray" type="button" name="button">.button.dark-gray</button>
<button class="button medium-gray" type="button" name="button">.button.medium-gray</button>
